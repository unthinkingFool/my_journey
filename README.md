##  About This Repository

This repository is my **living preparation log** for technical job interviews, targeted at roles in **AI/ML Engineering**, **LLM Systems**, and **Applied Machine Learning**. Every day I study, I push an update — tracking concepts learned, code written, and problems solved.

The roadmap spans everything from core AI/ML fundamentals to system design, DSA, databases, agentic AI, and production-grade DevOps — a full-stack preparation approach for a modern engineering role.

---

##  Preparation Roadmap

### 1. Artificial Intelligence & Machine Learning *(Core Focus)*
My primary domain. Covers both the theoretical foundations and hands-on applied work.
- Machine Learning fundamentals (supervised, unsupervised, reinforcement learning)
- Deep Learning, neural networks, transformers, and LLMs
- Model evaluation, fine-tuning, and deployment
- RAG (Retrieval-Augmented Generation) pipelines
- Vector databases, embeddings, and semantic search
- **Technologies:** Python, PyTorch, HuggingFace, FAISS, ChromaDB, CLIP

---

### 2. Agentic AI & LLM Frameworks
Building intelligent, multi-step AI systems using modern agentic tooling.
- LangChain — chains, agents, tools, memory
- LangGraph — stateful multi-agent workflows
- Multi-agent pipeline design and orchestration
- Prompt engineering, tool use, and agent evaluation
- **Technologies:** LangChain, LangGraph, Groq, Tavily, Streamlit

---

### 3. Data Structures & Algorithms *(C++)*
Rigorous DSA preparation for coding rounds.
- Arrays, linked lists, trees, graphs, heaps, tries
- Sorting, searching, dynamic programming, greedy, backtracking
- Graph algorithms (BFS, DFS, Dijkstra, MST)
- Time/space complexity analysis
- **Language:** C++

---

### 4. Databases & SQL
Understanding both relational and non-relational data systems with hands-on practice.
- Relational databases — schema design, normalization, indexing
- SQL — joins, aggregations, window functions, query optimization
- NoSQL concepts — document stores, key-value, wide-column
- Transactions, ACID properties, CAP theorem
- **Technologies:** PostgreSQL, MySQL, SQLite

---

### 5. System Design
Designing scalable, fault-tolerant systems end to end.
- Low-Level Design (LLD) — OOP, design patterns, SOLID principles, building components from scratch
- High-Level Design (HLD) — distributed systems, scalability, load balancing, caching
- Real-world system design case studies (URL shortener, search engine, streaming service, etc.)
- API design and microservices architecture

---

### 6. CS Fundamentals
The foundations that underpin everything else.
- **Operating Systems** — processes, threads, memory management, scheduling, concurrency
- **Computer Networking** — TCP/IP, HTTP/HTTPS, DNS, sockets, REST
- **Database Internals** — storage engines, B-trees, WAL, query planning

---

### 7. DevOps & Containerization *(Planned)*
Taking projects from local to production-ready.
- Docker — containerization, images, volumes, networking
- Docker Compose for multi-service applications
- CI/CD pipelines basics
- Infrastructure as Code (IaC) concepts
- **Technologies:** Docker, GitHub Actions

---

### 8. Cloud Technologies *(Planned)*
Deploying and managing applications at scale in the cloud.
- Cloud fundamentals — compute, storage, networking
- Serverless, managed services, and MLOps on cloud
- **Platforms:** AWS / Azure
